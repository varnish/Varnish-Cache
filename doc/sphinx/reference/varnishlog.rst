==========
varnishlog
==========

--------------------
Display Varnish logs
--------------------

:Manual section: 1

SYNOPSIS
========

.. include:: ../include/varnishlog_synopsis.rst
varnishlog |synopsis|

OPTIONS
=======

The following options are available:

.. include:: ../include/varnishlog_options.rst

SIGNALS
=======

* SIGHUP

  Rotate the log file (see -w option)

* SIGUSR1

  Flush any outstanding transactions
